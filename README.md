# Ra-zesDigitais-Agrinho2026
O projeto é um aplicativo web focado na rastreabilidade de alimentos orgânicos em Candói/PR. Ao escanear um QR Code, o usuário visualiza dados reais do produto: produtor, localização, datas de plantio/colheita, validade e manejo sustentável.O sistema inclui recursos de acessibilidade e um jogo educativo sobre colheita saudável.
