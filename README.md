# Ra-zesDigitais-Agrinho2026

O Projeto🌱
--
Criação: Aplicativo web interativo criado pela aluna Ana Giulia Solek Machado, da Escola Estadual Santa Clara.

Propósito: Desenvolvido especialmente para o Concurso Agrinho 2026 para unir tecnologia e campo, valorizando os produtores de Candói, no Paraná.

Funcionamento Geral: Uma plataforma de rastreabilidade onde o consumidor descobre a origem exata e o histórico do alimento orgânico que comprou.

O Jogo Integrado: Inclui um minijogo educativo de colheita, no qual o jogador move uma cesta para capturar alimentos saudáveis e desviar de agrotóxicos, ensinando sustentabilidade de forma rápida e divertida.

O Grande Objetivo📃
--
Conexão Direta: Ligar quem planta a quem consome, promovendo total transparência e consumo consciente.

Valorização Local: Fortalecer a agricultura familiar ao dar visibilidade ao trabalho e à localização dos pequenos produtores da região.

Segurança Alimentar: Garantir que o comprador conheça detalhes essenciais do produto, como datas de plantio, colheita e período de validade.

Conscientização: Ensinar a comunidade de forma lúdica sobre a importância de escolher alimentos limpos e os riscos da contaminação química.

Cliques e Leituras (Como Funciona)
--
🔎Botão Inicial: Ativa a câmera do dispositivo para iniciar o escaneamento do produto orgânico.

🔁Botão de Retorno: Limpa a tela após exibir as informações do alimento e prepara o sistema para uma nova leitura.

 ➕/➖Botões de Fonte: Aumentam ou diminuem o tamanho do texto na tela de forma dinâmica, facilitando a leitura.

🌓Botão de Alto Contraste: Muda as cores do site para preto, branco e amarelo para ajudar pessoas com dificuldades visuais.

🔄Botão do Jogo: Reinicia o placar e começa uma nova partida no minijogo da colheita saudável.

📷Leitura pela Câmera: O usuário aponta a câmera do celular ou computador diretamente para o QR Code físico na embalagem do produto.

📁Leitura por Código Digital (Arquivo): Permite carregar a imagem do QR Code salva na pasta do próprio dispositivo caso a câmera não esteja disponível.

De Onde Veio Tudo Isso? (Fontes)
--
Origem Tecnológica: Toda a engenharia de software e a lógica do sistema foram geradas em colaboração com o Gemini, a inteligência artificial do Google.

Suporte Visual: As imagens profissionais de alta resolução exibidas na tela de resultados dos produtos foram obtidas diretamente da plataforma Pinterest.

Prevenção de Erros: As fotos do Pinterest servem como um plano de segurança caso ocorra qualquer falha no carregamento dos arquivos de imagem locais.

O Design e os Códigos
--
Paleta de Cores e Estilo: Visual moderno com tons de verde pastel e elementos arredondados que remetem à natureza e à sustentabilidade.

Responsividade: O design se adapta perfeitamente a qualquer tamanho de tela, funcionando bem tanto em computadores quanto em celulares.

Estrutura com HTML (index.html): Código responsável por montar o esqueleto do site, organizando os textos, as seções e onde cada botão deve ficar.

Estilização com CSS (style.css): Código que dá vida ao visual, aplicando as cores, as fontes personalizadas, o layout de tela dividida e os efeitos de acessibilidade.

Inteligência com JavaScript (script.js): Código que faz a mágica acontecer, controlando o leitor de QR Code (seja por câmera ou arquivo), processando o banco de dados e rodando o jogo da colheita.
